Open Scalping

Alpha released

Github Preview

Open Scalping is a stock exchange simulation platform. With it the user can define a custom asset and simulate a Bovespa B3 stock exchange market with bot players. At the simulated stock exchange environment the user can do assets purchase and sold operations, and use the most common flow analysis stock enchange market operating tools, like the deals history, offers book, DOM and volume-at-price. A candlestick graphic is going to be introducet in a future release. The application is being developed with Angular 8.
